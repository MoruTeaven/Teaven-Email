## 设计变更摘要

| 维度 | 之前 | 之后 |
|------|------|------|
| 色彩模式 | 仅深色 | 深色/浅色双模式，localStorage 持久化 |
| 图标方案 | Font Awesome (CDN) | 内联 SVG（零外部依赖） |
| 导航栏 | 固定宽 280px | 可折叠 260px ↔ 72px |
| 移动端 | 侧边栏隐藏 | 抽屉式菜单 + 遮罩层 |
| 主题切换 | 无 | 右上角一键切换，跟随系统 |
| 顶部栏 | 无 | 面包屑 + 刷新 + 主题切换 |
| 数据展示 | 基础列表 | 统计卡片 + 环形图 + 柱状图 + 时间线 |
| 表格 | 简单列表 | 搜索/筛选/排序/分页/批量操作 |
| 表单 | 基础输入 | 表单验证/开关/动态表单/帮助说明 |

## 8个功能模块

1. **仪表盘** — 统计卡片 + 资源概览 + 活动时间线 + 趋势图
2. **用户管理** — CRUD + 搜索筛选排序分页 + 模拟登录
3. **发送通道** — SMTP/API/Cloudflare 动态表单 + 状态管理
4. **发件账号** — 用量进度条 + 测试发送
5. **分类路由** — 规则管理
6. **发送日志** — 日志列表
7. **数据分析** — 环形图 + 趋势图
8. **系统设置** — 配置 + 安全 + 信息
